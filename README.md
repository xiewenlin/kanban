# kanban
#专注于提高办公效率的极速在线简易敏捷看板。使用地址：https://xiewenlin.com/kanban/ <br>
### 优点：<br>
#### 1.在线、极速、简易的敏捷看板<br>
#### 2.技术栈：Java8 + Netty4 + Vue.js<br>
#### 3.基于精益创业思想<br>
### linux后台运行部署命令：<br>

nohup java -jar kanban.jar --server.port=9000 >consoleMsg.log 2>&1 & <br>
